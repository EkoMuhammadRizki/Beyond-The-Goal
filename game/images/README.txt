IMAGE FILES PLACEHOLDER

Game ini menggunakan sistem placeholder otomatis.
Jika file image tidak ditemukan, game akan menggunakan warna solid sebagai placeholder.

STRUKTUR FOLDER:
- characters/  : Character sprites (300x600px PNG)
- backgrounds/ : Background images (1920x1080px PNG/JPG)

Untuk mengganti placeholder dengan asset asli:
1. Buat atau siapkan image sesuai spesifikasi
2. Simpan dengan nama file yang sesuai
3. Letakkan di folder yang tepat
4. Game akan otomatis menggunakan image baru

Lihat ASSET_GUIDE.md untuk daftar lengkap file yang dibutuhkan.
