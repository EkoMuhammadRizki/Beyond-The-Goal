## ⚡ Quick Start (3 Langkah)

### 1️⃣ Download Ren'Py SDK

**Link:** https://www.renpy.org/latest.html

- Pilih versi untuk sistem operasi Anda
- Download dan extract
- Jalankan `renpy.exe` (Windows) atau `renpy.sh` (Mac/Linux)

**Waktu:** ~5 menit

---

### 2️⃣ Load Project

Di Ren'Py Launcher:

1. Klik **"+ Add Existing Project"**
2. Pilih folder **"Beyond The Goal"**
3. Klik **"Select Folder"**

**Waktu:** ~30 detik

---

### 3️⃣ Launch Game

1. Pilih project **"Beyond The Goal"** di list
2. Klik **"Launch Project"**
3. **SELESAI!** Game akan terbuka dan siap dimainkan! 🎉

**Waktu:** ~10 detik

## ❓ FAQ Cepat

### Q: Apakah game ini gratis?
**A:** Ya! 100% gratis untuk dimainkan dan dimodifikasi.

### Q: Apakah perlu asset tambahan?
**A:** Tidak! Game sudah lengkap dengan placeholder. Asset tambahan opsional.

### Q: Berapa lama waktu bermain?
**A:** Sekitar 30-45 menit untuk satu playthrough lengkap.

### Q: Apakah ada multiple endings?
**A:** Ya! Ending tergantung pada player style Anda.

### Q: Bisa dipakai untuk pembelajaran?
**A:** Tentu! Game ini dirancang untuk edukasi sepak bola dan character development.

## 💻 Untuk Developer

### Explore Code
1. Buka file `.rpy` dengan text editor
2. Baca comments di code
3. Pahami struktur modular
4. Coba modifikasi kecil

### Struktur File
```
game/
├── script.rpy          # Main entry
├── chapter1.rpy        # Chapter 1
├── chapter2.rpy        # Chapter 2
├── chapter3.rpy        # Chapter 3
├── characters.rpy      # Characters
├── variables.rpy       # Variables
└── custom_screens.rpy  # UI
```

### Edit & Test
1. Edit file `.rpy`
2. Save
3. Press **Shift+R** di game (reload)
4. Test perubahan

**Panduan:** [QUICKSTART.md](QUICKSTART.md) & [CONTRIBUTING.md](CONTRIBUTING.md)
